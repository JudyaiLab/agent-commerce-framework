## 商業模型摘要

```
免費註冊 + 教學 → 買 MCP Commerce Builder ($199-999)
→ 架設付費 MCP Server → 上架 AgenticTrade → 平台 10% 抽成
→ 忠誠降佣（10% → 8% → 6% → 5%）留住賣家
```

**雙引擎營收：** Builder 銷售（近期可控）+ 平台抽成（長期成長）

**已完成的內容資產：**

- Blog 大綱（Agent 經濟時代）

### 2. 興趣（Interest）

**目標：註冊轉換率 10%+**

| 策略 | 細節 |
|------|------|
| **免費教學系列** | 「30 天 Agent 商業化挑戰」— 從零到上架 |
| **免費 Builder 試用** | `--payment none` 生成免費版，體驗後升級 |
| **CoinSifter 免費 tier** | 100 次/月免費 API 調用，展示平台價值 |
| **Email 自動化** | Resend + Brevo：註冊 → 歡迎信 → 3 天後教學 → 7 天後 Builder 介紹 |

### 3. 轉換（Conversion）

**目標：Builder 購買轉換率 3-5%**

| 策略 | 細節 |
|------|------|
| **價值對比** | 「自己做 vs 用 Builder」→ Builder 省 40+ 小時 |
| **RapidAPI 遷移** | 「從 25% 降到 10%」— 針對現有 API 供應商 |
| **限時優惠** | 首 100 名 Builder 購買者享早鳥價（考慮 $149） |
| **案例研究** | CoinSifter 做為第一個成功案例 |
| **社會證明** | Early Adopter 徽章 + 推薦人數統計 |

### 4. 留存（Retention）— 品質等級制 v4

**目標：6 個月留存率 60%+**

→ 詳見 [整合方案 v4](../../ai-logs/proposals/referral_loyalty_integrated_v4.md)

核心機制（Judy 2026-03-22 核准）：
- 新手免費期 90 天（0% 佣金）
- 品質決定佣金（不是銷售額決定）：
  - 🟢 Standard：10%（預設）
  - ⭐ Verified Agent：8%（穩定 ≥99% + 回應 <2s + 30 天在線）
  - 👑 Premium Agent：6%（穩定 ≥99.5% + 回應 <500ms + 90 天在線）
- Builder 終身免費更新 + 月度模板投放
- Founding Seller（前 50 名）永久徽章 + 佣金上限 8%
- 品質下降自動降級（有動力維護服務）
- 離開平台品質數據歸零（沉沒成本 = 黏著度）

### 5. 推薦（Referral）— 持續分潤制 v4

**目標：20% 新用戶來自推薦**

| 策略 | 細節 |
|------|------|
| **推薦開店（持續分潤）** | 推薦人永久拿平台收入的 20%（平台抽 10%→推薦人 2%，平台留 8%） |
| **推薦買 Builder** | 推薦人得 $30（15%），朋友享 $20 折扣 |
| **防作弊** | 被推薦人需上架+健康檢查+首筆 $10+ 收入才啟動 |
| **成本控制** | 推薦分潤跟品質等級連動（品質降佣越多，推薦分潤自動降），平台最低淨賺 3.8% |

### 關鍵學習

1. **套裝策略有效**：$59 套裝比 $14.90 單品 AOV 高 4x，15.4% 升級率
2. **亞太華人市場是基本盤**：90% 客戶來自中文區，英文市場待開發
3. **Blog 導流可行**：judyailab.com 帶來 10% 轉換，SEO 投資有回報
4. **零退款 = 產品品質過關**：17 筆交易零退款，QA 流程有效
5. **月營運成本極低**：$20/月（MiniMax 訂閱），幾乎零 CAC

### 基於數據的行銷調整

- **加碼英文內容**：當前 90% 亞太客戶，英文市場有 10x 成長空間
- **推動套裝銷售**：15.4% 升級率可提升至 25%+（更醒目的 CTA + 比較表）
- **強化 Blog SEO**：目前帶 10% 流量，目標提升至 30%
- **上架更多 $100+ 產品**：目前 AOV $21.68，需推高至 $50+
