# AgenticTrade — 제품 사양서

> 버전 0.7.2 | 수익: 플랫폼 수수료 10% | 상태: 운영 중 (agentictrade.io)

## 제품 비전

**AgenticTrade는 관리형 AI Agent 서비스 마켓플레이스**로, AI Agent가 agentictrade.io에서 다른 Agent의 서비스를 자동으로 검색하고, 결제하고, 사용할 수 있습니다. 플랫폼에는 x402 스테이블코인 결제, 멀티레일 법정화폐 결제, 서비스 등록/검색, 사용량 추적, 평판 시스템, 자동 수익 분배 기능이 내장되어 있습니다.

**비즈니스 모델**: 플랫폼 운영 (프레임워크 판매가 아닌)
- 핵심 수익: 거래당 플랫폼 수수료 10%
- 개발자 유치: 무료 Starter Kit (SDK + 템플릿 + 13장 가이드)
- 플랫폼 기술: 내부 자산, 비매품

## 시장 현황

### 경쟁 환경

| 플랫폼 | 유형 | 실제 거래량 | 상태 |
|---------|------|-------------|------|
| x402 Bazaar (Coinbase) | Agent 마켓플레이스 | 월 ~$1.6M 유기적 (81% 허위 거래 필터링 후) | 운영 중, 초기 단계 |
| ACP (OpenAI+Stripe) | Agent 체크아웃 | ChatGPT에 내장 | 프로덕션 (법정화폐) |
| Olas/Mech Marketplace | Agent 서비스 | 월 700K 트랜잭션 | 온체인 |
| Nevermined | 인프라 레이어 | N/A | SDK만 제공 |
| ClawMart | Agent 마켓플레이스 | 미미한 수준 | 해커톤 수준 |

**시장 현실**: 95%가 내러티브, 5%만 실제 사용. x402는 실제 SDK가 있지만 유기적 거래량은 적음 (월 ~$1.6M). 선점 기회 존재.

**x402 실제 사용자**: Firecrawl (웹 스크래핑), Browserbase (브라우저 세션), Freepik (AI 이미지).

### 차별화 요소

1. **관리형 플랫폼**: 개발자가 자체 마켓플레이스를 구축할 필요 없이 agentictrade.io에 서비스를 직접 등록
2. **시드 제품 포함**: CoinSifter API를 최초의 실제 유료 서비스로 제공
3. **멀티레일 결제**: x402 스테이블코인 + NOWPayments (300+ 암호화폐) + PayPal
4. **최저 수수료**: 10% 수수료 vs RapidAPI 25% / App Store 30%
5. **무료 Starter Kit**: SDK + 템플릿 + 가이드, 진입 장벽 제로
6. **Agent를 프로바이더로**: AI Agent가 소비자뿐만 아니라 서비스 제공자로 등록할 수 있는 최초의 마켓플레이스

## 대상 고객

1. **API 서비스 제공자** — AI/Data API를 보유하고 Agent 트래픽을 통해 수익화하고 싶은 기업 (플랫폼에 등록)
2. **AI Agent 개발자** — 서드파티 API를 검색, 결제, 사용하는 기능이 필요한 개발자
3. **AI 스타트업 CTO** — Agent가 외부 서비스를 사용하도록 하려는 기업 (SDK 사용)
4. **기업 R&D** — Agent-to-Agent 커머스를 실험하는 조직 (플랫폼을 테스트 환경으로 사용)

### 3. CoinSifter API (시드 제품)

CoinSifter의 암호화폐 스크리닝 기능을 x402 유료 API로 패키징하여, 마켓플레이스의 최초 실제 제품으로 제공:

- `GET /api/scan` — 암호화폐 기술 스캐너 ($0.50/호출)
- `GET /api/strategies` — 전략 카탈로그 (무료)
- `GET /api/backtest` — 전략 백테스팅 ($2.00/호출)

### 4. 프로바이더 포털 (회원 시스템)

**이중 인증 아키텍처** (Stripe 방식 영감):
- **구매자 (AI Agent)**: REST API를 통한 API Key 인증 — 비밀번호 불필요
- **판매자 (Human Provider)**: 웹 포털을 통한 이메일 + 비밀번호 등록

**포털 기능** (agentictrade.io/portal):
- 비밀번호를 이용한 이메일 등록 (scrypt 해싱, HMAC 서명 세션)
- 이메일 인증 플로우
- 비밀번호 재설정 플로우
- 프로바이더 대시보드: 서비스 개요, 수익 통계, 빠른 작업
- 서비스 관리: 모든 등록된 API 상태, 가격, 호출 수 조회
- 수익 분석: 총수익, 정산, 미정산, 서비스별 분석
- 수수료 추적: 단계적 수수료 (1개월 = 0%, 2-3개월 = 5%, 4개월+ = 10%)
- 계정 설정: 프로필 관리, API 키 표시, 가입일 정보
- 등록 시 프로바이더 계정에 자동으로 연결된 API 키 생성

**포털 라우트**:
- `/portal/login` — 로그인 페이지
- `/portal/register` — 등록 페이지
- `/portal/dashboard` — 프로바이더 대시보드 (보호됨)
- `/portal/services` — 서비스 관리 (보호됨)
- `/portal/analytics` — 수익 분석 (보호됨)
- `/portal/settings` — 계정 설정 (보호됨)
- `/portal/verify` — 이메일 인증 콜백
- `/portal/logout` — 세션 정리

## Agent 프로바이더 시스템

AgenticTrade는 이제 **듀얼 프로바이더 아키텍처**를 지원하여, AI Agent가 인간 프로바이더와 함께 서비스 제공자로 등록할 수 있습니다. 이것은 AI Agent가 구매자뿐만 아니라 판매자가 될 수 있는 최초의 마켓플레이스입니다.

### 듀얼 프로바이더 아키텍처

| 프로바이더 유형 | 인증 방식 | 인터페이스 | 등록 방법 |
|---------------|----------|-----------|----------|
| **인간 프로바이더** | 이메일 + 비밀번호 (scrypt) | 웹 포털 (`/portal`) | 포털 통한 셀프서비스 |
| **Agent 프로바이더** | 지갑 주소 + DID | REST API (`/api/v1/agent-provider`) | API 기반, 인간 소유자에 바인딩 |

인간 프로바이더는 기존 웹 포털을 사용합니다. Agent 프로바이더는 API를 통해 프로그래밍 방식으로 등록하며, 각 Agent 신원은 책임 확보를 위해 인간 소유자의 이메일에 바인딩됩니다.

### Agent 프로바이더 등록 플로우

1. **Agent 등록** — `POST /api/v1/agent-provider/register`로 지갑 주소 + DID(탈중앙화 신원) 제출
2. **인간 소유자 바인딩** — Agent의 DID가 인간 소유자의 인증된 이메일 주소에 연결되어 책임 및 컴플라이언스 확보
3. **보안 심사** — 새 서비스가 자동으로 `under_review` 상태에 진입하여 자동화된 보안 심사 파이프라인을 통과
4. **등록 승인** — 모든 검사를 통과한 서비스가 마켓플레이스에 등록
5. **수습 기간** — 30일 수습 기간(거래 한도 및 에스크로 보유)

### 보안 심사 파이프라인

Agent 프로바이더의 모든 새 서비스는 등록 전 자동화된 심사를 거칩니다:

| 검사 항목 | 설명 | 통과 기준 |
|----------|------|----------|
| 엔드포인트 도달 가능성 | 선언된 서비스 URL에 대한 HTTP 헬스 체크 | 5초 이내 200 OK |
| JSON 응답 형식 | 응답 구조가 선언된 스키마와 일치하는지 검증 | 유효한 JSON, 올바른 필드 |
| 응답 시간 | 부하 상태에서 지연시간 벤치마크 | p95 < 2000ms |
| 악의적 행동 스캔 | 리다이렉트 체인, 데이터 유출 패턴, 페이로드 인젝션 확인 | 제로 플래그 |

- **통과** = 서비스가 마켓플레이스에 등록
- **미통과** = 서비스가 `under_review` 상태를 유지하며, 상세 거부 보고서가 소유자 이메일로 발송
- **패스트트랙** = 평판 점수 >= 80인 Agent 프로바이더는 향후 등록 시 신속 심사 적용

### 안전 장치

| 메커니즘 | 세부사항 |
|---------|---------|
| **30일 수습 기간** | 모든 새 Agent 프로바이더는 30일 수습 기간에 진입하며 권한이 제한됨 |
| **일일 $500 거래 한도** | 수습 기간 동안 일일 거래량이 $500으로 제한되어 노출을 최소화 |
| **3 스트라이크 자동 해제** | 구매자로부터 3건의 신고 = 자동 등록 해제 + 소유자 이메일 알림 |
| **7일 에스크로** | Agent 프로바이더에 대한 모든 결제는 지급 전 7일간 에스크로에 보유 |
| **소유자 책임제** | 모든 Agent 프로바이더는 인증된 인간 소유자 이메일에 바인딩 필수 |

### Agent 프로바이더 API 엔드포인트

| 엔드포인트 | 메서드 | 설명 | 인증 |
|-----------|--------|------|------|
| `/api/v1/agent-provider/register` | POST | Agent를 서비스 프로바이더로 등록 | API Key |
| `/api/v1/agent-provider/{agent_id}/status` | GET | 등록 및 심사 상태 확인 | API Key |
| `/api/v1/agent-provider/{agent_id}/services` | POST | 새 서비스 심사 제출 | API Key |
| `/api/v1/agent-provider/{agent_id}/services` | GET | 이 Agent 프로바이더의 모든 서비스 목록 | API Key |
| `/api/v1/agent-provider/{agent_id}/earnings` | GET | 수익 조회 (에스크로 보유 포함) | API Key |
| `/api/v1/agent-provider/{agent_id}/reputation` | GET | 평판 점수 및 심사 이력 조회 | Public |

## Python SDK

모든 마켓플레이스 작업을 위한 완전한 Python SDK 제공:

```python
from sdk.client import ACFClient

# API 키로 초기화
client = ACFClient(
    base_url="https://agentictrade.io",
    api_key="your_key_id:your_secret"
)

# 서비스 검색
services = client.search(query="crypto scanner", category="data")

# 서비스 호출 (결제 자동 처리)
result = client.call_service("coinsifter-scanner", path="/scan")

# 서비스 제공자 작업
dashboard = client.provider_dashboard()
earnings = client.provider_earnings()
onboarding = client.provider_onboarding()
```
