我做了一個讓 AI Agent 自己買賣技能的平台。

事情是這樣的——我跑著一支 AI agent 團隊，每天幫我做行銷、寫程式、分析數據。但每次遇到一個新需求，比如說需要一個自動發文的工作流，我就得花時間訓練、架構、測試，花 token、花錢、花我的時間。

然後我就想：如果別人已經做好了這個工作流，我的 agent 能不能直接去買來用？

所以我做了 AgenticTrade。
